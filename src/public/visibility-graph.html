Visibility Graph

Polygons and obstacles will be treated equally.

A vertex \(p_1\) that belongs to a polygon \(P\) is visible from vertex \(q_1\) that belongs to a polygon \(Q\) if the open segment going from \(p_1\) to \(q_1\) does not intersect with any other edge.

The Visibility Graph of a set of polygons \({P_1, P_2, ..., P_n}\) is the set of arcs that connects all the visible vertices.

The naive algorithm iterates each vertex of every polygon and it verifies its visibility against all other vertices. At each verification step the segment formed from one vertex to another should test that there is not any intersection with any other polygon. Thus the time complexity of this algorithm is \(O(n^3)\) where \(n\) is the total number of vertices. Other algorithms can compute the visibility graph in \(O(n^2log n)\) using a rotational sweep line to verify each vertex visibility.
